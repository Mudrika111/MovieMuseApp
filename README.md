# 🎥 MovieMuse

MovieMuse is your smart, stylish movie recommender — built with Python and Streamlit.

## ✨ Features
- 🎬 Personalized movie recommendations  
- 🔥 Trending movies from TMDB  
- 🎲 “Feeling Lucky” feature  
- 🎨 Beautiful UI in pastel pink and blue
